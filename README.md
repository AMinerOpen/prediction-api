# prediction-api

## 环境配置

推荐使用`anaconda`配置运行环境

### windows/linux

```bash
conda install -c mbednarski fasttext
conda install -c anaconda scikit-learn
conda install -c conda-forge jieba
conda install -c conda-forge requests
```

### osx

```bash
conda install -c conda-forge fasttext
conda install -c anaconda scikit-learn
conda install -c conda-forge jieba
conda install -c conda-forge requests
```
